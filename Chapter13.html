When many duplicated CSS declarations apply to the same element, the web browser picks up only one style and applies it to the page.

Therefore, we will see the CSS line in blue as the last declaration is "span {color: darkblue}".

Inheritance in CSS

In CSS, the parent element’s styles (like text color) are inherited by its children automatically. However, not all properties are inherited, such as the “border” property.
