The layout is the way HTML elements are placed on a web page. We often use CSS to build a design by arranging the elements into our preferred visual structure.

The default value of the CSS “ position ” property is “ static .” The element is placed where it should be inside the web page according to its order in the HTML code.An element with “ position: relative ” is relative to its normal position. We use the top, right, bottom , and left properties to set the elements position.
