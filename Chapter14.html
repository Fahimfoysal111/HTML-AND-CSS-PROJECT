Box Sizing is the default CSS applied to all HTML elements in our web pages. However, this CSS rule confuses developers since the actual size is not as the "width" and "height" declarations.

We have four CSS properties for the top, right, bottom, and left sides:

  • margin-top
  • margin-right
  • margin-left
